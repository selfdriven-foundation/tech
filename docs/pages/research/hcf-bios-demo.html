S01E01 — IBM BIOS ROM REVERSE ENGINEERING SIMULATOR

This interactive demo replicates what Cardiff and MacMillan are doing with that light board in the show.


THE SETUP

  • They've tapped directly into the IBM PC's BIOS ROM chip
  • Each LED on the board = one bit on the data or address bus
  • ON = 1, OFF = 0 — binary at the electrical level
  • Groups of 8 LEDs = 1 byte of firmware data

THE TRANSLATION

Every byte goes through three representations:

1010 1100 → AC → MOV AX, 0xAC00

  • Binary: what the chip outputs electrically
  • Hex: compact shorthand (1 byte = 2 hex digits)
  • Assembly: human-readable machine code

THE GOAL

Clean-room reverse engineering — understand the BIOS well enough to clone it without directly copying it. This is exactly how Compaq legally cloned the IBM PC in 1982.
