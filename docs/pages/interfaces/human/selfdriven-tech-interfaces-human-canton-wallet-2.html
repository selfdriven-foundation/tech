Setup Required
Serve via localhost
WebAuthn / Passkeys require a secure context — either https:// or http://localhost. Opening directly as file:// won't work.

Run one command below, then open http://localhost:8080/selfdriven-canton-wallet.html in your browser:
Python 3 — built-in, no install needed
$ cd /path/to/folder && python3 -m http.server 8080
npm install -g http-server (once)
$ http-server . -p 8080 --cors
No install needed
$ npx serve . -l 8080
Ruby — built-in
$ ruby -run -e httpd . -p 8080
PHP — built-in
$ php -S localhost:8080
→ http://localhost:8080/selfdriven-canton-wallet.html
Demo Mode — passkey flows are simulated. Serve via localhost for real biometric authentication.
